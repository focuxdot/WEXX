# wexx

在微信里,给你电脑上的 Codex 派活。

扫一次码,之后随时随地在手机微信上发消息:整理资料、处理文件、写文档、跑脚本——任务在你自己的电脑上执行,结果推回微信。

## 它是怎么工作的

wexx 是一个 Codex skill + 本地常驻服务:

```
手机微信 ⇄ 腾讯官方 iLink Bot 通道 ⇄ 你的电脑(wexx)⇄ 本机 Codex
```

- **没有第三方服务器**:消息经腾讯官方 Bot API 直连你的电脑,本项目没有自己的后端;
- **不碰你的 Codex 配置**:不读写 `config.toml`、不涉及任何模型或 API key,用的就是你已经配好的 Codex;
- **数据全在本地**:会话、文件都只存在你自己的机器上;
- 微信里的对话会同时出现在 Codex 桌面版的会话列表里,回到电脑可以无缝接着聊。

## 安装

需要:macOS、已安装并配置好的 Codex(CLI 或桌面版)、Node.js 18+。

**方式一:让 Codex 自己装。** 把这句话发给 Codex:

> 帮我安装 wexx:https://github.com/focuxdot/WEXX

**方式二:在终端里粘贴一行:**

```bash
git clone https://github.com/focuxdot/WEXX.git ~/.codex/skills/wexx && cd ~/.codex/skills/wexx && npm install
```

装好后,在 Codex 里输入 `$wexx`(或 `/wexx`),扫码即完成绑定(首次会请你确认一次运行模式,只问这一次)。后台服务自动启动,并设为开机自启。

## 使用

绑定后,在微信里发消息就是派活。第一条消息由你主动发起,你会收到问候语,任务随即开始执行。

四条指令由本地程序直接处理,不会发给 Codex:

| 发送 | 作用 |
|---|---|
| `/stop` 或 `停` | 立即中断当前任务 |
| `/new` 或 `新任务` | 开启全新会话(不带旧上下文) |
| `/status` 或 `状态` | 查看任务状态 |
| `/help` 或 `帮助` | 查看帮助 |

其余任何消息都会原样交给 Codex 执行。

管理连接:在 Codex 里再次输入 `$wexx` 可查看状态(后台服务掉线会自动拉起);想撤销就对 Codex 说"解绑微信"。

## 安全须知

- 只有**扫码绑定的那个微信号**能与你的 Codex 对话;
- 任务默认**全速执行**(不逐条审批,可读写文件和联网)——这是"派活"体验的前提,也是你首次绑定时明确确认过的选择,不是静默默认;
- 因此请只派你信任内容范围内的活:任务接触的网页和文件可能包含恶意指令(提示注入),全速模式下没有人工审批兜底;
- 随时有刹车:微信里发 `停` 立即中断;
- 逐条审批的"严格模式"在路线图中(v1.1)。

## 已知限制

- 电脑休眠或关机时服务不可用(消息不会丢,唤醒后可继续)——所有本地方案的共同限制;
- 开机自启目前只支持 macOS;
- 产出的文件保存在本机(`Documents/Codex/` 下)并告知路径,文件/图片直接回传微信在路线图中(v1.0.1);
- 面向"派活"场景(通用任务),不是在微信里做项目级远程开发。
